# NLP Text Processing
Este repositorio contiene prácticas académicas de procesamiento de lenguaje natural enfocadas en el tratamiento y representación de texto.

## Contenido

- Extracción de texto desde páginas web de noticias
- Limpieza y normalización de texto
- Tokenización
- Eliminación de stopwords
- Lematización
- Representaciones de texto:
  - Frecuencia de términos
  - Binarización
  - TF-IDF
- Medidas de similitud:
  - Similitud coseno
- Métodos probabilísticos:
  - Probabilidad condicional

Los ejemplos utilizan datasets en formato CSV generados a partir de textos de noticias públicas.
